# 🚀 深圳技术大学飞跃手册 (SZTU Feiyue)

欢迎！**深圳技术大学飞跃手册**是一份汇聚深圳技术大学往届生留学、保研、考研、就业四个大方向的案例与经验分享网站。

## 🎉 写在前面

这份手册旨在打破信息差，帮助学弟学妹们少走弯路。无论你的目标是升学还是就业，在这里都能找到前人的足迹。
